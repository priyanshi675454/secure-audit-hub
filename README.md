## ğŸ¯ How It Works

### **6-Step Official Process**

Based on the official Solana Audit Subsidy Program requirements:

#### **Step 1: Application (Day 1)**
**What happens:**
- Submit application via **TWO platforms**:
  1. Subsidy Program Application Form (this website)
  2. Earn platform (earn.superteam.fun)
- Applications automatically match between platforms
- Add website or X profile in "Link to Submission" field on Earn

**What you need:**
- Project name and description
- GitHub repository
- Team information
- Expected launch date
- Contact email

#### **Step 2: Assessment (1-2 weeks)**
**What happens:**
- Expert panel reviews application
- Panel: Superteam, MonkeDAO, Jito, and Areta
- Evaluation against standardized rubric
- Technical code quality assessment
- Team viability check
- Budget determination

**What you should do:**
- Check email regularly
- Respond to panel questions promptly
- Maintain communication

#### **Step 3: Approval & Subsidy (Day 15-20)**
**What happens:**
- Receive approval notification
- Get assigned subsidy amount ($5k - $50k)
- Receive Subsidy Voucher Code
- Onboarding to marketplace
- Access to all 15 approved providers

**What you get:**
- Approval email
- Unique voucher code
- Marketplace dashboard access

#### **Step 4: Marketplace (3-5 days)**
**What happens:**
- Browse all 15 approved audit providers
- Receive offers from auditors
- Compare pricing, timelines, and services
- Review provider specializations
- Read past work and reviews

**Use this platform to:**
- Compare providers side-by-side
- Check ratings and reviews
- View detailed provider profiles
- Contact providers directly

#### **Step 5: Quote Selection (Day 25-30)**
**What happens:**
- Select your preferred provider
- Review final quote and timeline
- **Apply subsidy to any offer**
- Get up to 30% coverage (max $50k)
- Sign audit agreement

**How subsidy works:**
- 30% automatic discount on any provider
- Example: $40k audit â†’ $28k after subsidy
- Maximum subsidy: $50k
- Voucher must be used within program timeframe

#### **Step 6: Voucher Redemption**
**What happens:**
- Apply Subsidy Voucher Code at checkout
- Automatic 30% discount applied
- Begin audit process with selected provider
